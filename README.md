# Bases de Datos · Resumen interactivo (UTN.BA)

Resumen interactivo de **Bases de Datos (GDD)** de UTN.BA: estructuras de datos,
acceso a datos y teoría de bases de datos, con diagramas y visualizaciones animadas
para repasar para el parcial.

> Sin instalación, sin build, sin dependencias: HTML + CSS + JavaScript vanilla.
> Abrí `index.html` con doble clic y listo.

## 📚 Contenido

**Estructuras**
1. Grafos
2. Estructuras de Datos
3. Árboles
4. Árboles de Búsqueda
5. Compresión (Huffman)
6. Métodos de Ordenamiento

**Acceso a datos**
7. Hashing
8. Árbol B y B+

**Bases de datos**
9. Teoría de BD
10. Objetos de BD
11. Data Warehouse

**★ Examen**
- Repaso para el parcial

## 🚀 Cómo usarlo

- **Local:** abrí `index.html` en cualquier navegador (doble clic).
- **Online:** subilo a GitHub y conectalo a Vercel (framework: **Other**, sin build). Listo.

## 🗂️ Estructura del proyecto

```
index.html     · apunte completo (esqueleto + contenido + visualizaciones)
styles.css     · estilo visual
favicon.svg    · ícono del sitio
```

El diseño (sidebar, navegación, tarjetas, barra de progreso y menú móvil) sigue el
estilo del [template de apuntes](https://github.com/ejdavenheimer) para que todas las
materias **se vean iguales**.
